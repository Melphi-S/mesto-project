# Проект: Mesto

Одностраничный сайст, проект для второго спринта на курсе "Веб-разработчик плюс" от Яндекс.Практикум.

## Назначение

Учебный проект студента для отработки навыков в адаптивной вёрстке с HTML и CSS:

* flexbox;
* grid;
* медиазапросы;
* формы;
* модальные окна.
